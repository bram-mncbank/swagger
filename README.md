# Assesment 2: Swagger on Golang
## How to install
~~~
git clone https://github.com/bram-mncbank/swagger
~~~

## How to Run
### Run Application
Enter directory and run the main.go:
~~~
cd swagger
go run main.go
~~~
### Access the web and API
1. Open any browser and access *http://localhost:8181/swagger*
